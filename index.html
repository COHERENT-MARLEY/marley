MARLEY documentation

The source code for MARLEY v1.0.0 is available here or from the official MARLEY GitHub repository. Build requirements: g++ >= 4.9 (or clang++ >= 3.4) and GNU make

MARLEY may also be built as a product for the relocatable ups tool developed by Fermilab. To do so, use the code found in the ups-specific repository.

The nuclear structure data that we recommend for use with MARLEY are taken from TALYS-1.6 and are available in the source code tar.gz file under the structure/ subdirectory.

Documentation describing the new configuration file format for v1.0.0 is in preparation. The source code and the beginning of a draft user manual for v0.9.5 are both available in the meantime.
